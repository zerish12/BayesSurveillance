
# BayesSurveillance

**Bayesian Surveillance Methods for Healthcare Performance Monitoring**

`BayesSurveillance` provides Bayesian surveillance methods for
prospective monitoring of healthcare performance, patient safety, and
clinical quality indicators.

The package includes beta-binomial monitoring for binary outcomes,
gamma-Poisson monitoring for count outcomes, Bayesian CUSUM and EWMA
charts, posterior predictive alert probabilities, risk-adjusted
surveillance, calibration, ROC analysis, and research-quality graphical
summaries.

## Installation

``` r
# install.packages("remotes")
remotes::install_github("zerish12/BayesSurveillance")
```
